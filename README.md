## laket-admin 通用后台管理系统


### 项目介绍

*  `laket-admin` 是基于 `thinkphp6` 版本的后台快速开发框架


### 环境要求

 - PHP >= 7.1.0
 - thinkphp ^6.0.0
 - Fileinfo PHP Extension


### 特别鸣谢

感谢以下的项目,排名不分先后

### 开源协议

*  `laket-admin` 遵循 `Apache2` 开源协议发布，在保留本系统版权的情况下提供个人及商业免费使用。
